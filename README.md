# 🌿 Projeto Agrinho — Site Oficial

Site educacional do **Projeto Agrinho**, desenvolvido para o programa socioeducacional do Sistema FAEP/SENAR-PR no Paraná.

## 📁 Estrutura do projeto

```
agrinho/
├── index.html    ← Estrutura e conteúdo da página
├── style.css     ← Estilos, cores, layout e responsividade
├── script.js     ← Interatividade, animações e formulário
└── README.md     ← Este arquivo
```

## 🚀 Como usar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/agrinho.git
   cd agrinho
   ```

2. **Abra no navegador:**
   Basta abrir o arquivo `index.html` diretamente em qualquer navegador moderno.
   Não é necessário servidor local.

3. **Ou use o Live Server (VS Code):**
   Instale a extensão *Live Server* e clique em **"Go Live"** no `index.html`.

## ✨ Funcionalidades

- **Hero animado** com partículas de sementes em canvas
- **Navbar responsiva** com scroll inteligente e menu hamburguer
- **Seções:** Sobre, Temas, Galeria, Participar e Contato
- **Formulário de inscrição** com validação em JavaScript
- **Animações de entrada** via Intersection Observer
- **Design responsivo** para mobile, tablet e desktop
- **Acessibilidade:** foco visível, aria-labels, motion-reduce

## 📤 Publicar no GitHub Pages

1. Vá em **Settings → Pages** no seu repositório
2. Em *Source*, selecione `main` e pasta `/root`
3. Clique em **Save** — o site ficará disponível em:
   `https://seu-usuario.github.io/agrinho`
